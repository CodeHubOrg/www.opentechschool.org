OpenTechSchool to build first Learning Space

Team Berlin is set to build the first open tech learning space based on the values of OpenTechSchool. The goal is to run the full-time code learning programmen and master class for advanced programmers during the day and host more OpenTechSchool Workshops and Learning events during the evenings and weekends. To make this vision a reality, we need your support. Please consider donating to our crowd funding campaign or become an advocate for it.
